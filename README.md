# USED MARKET | 중고마켓

프로젝트 소개
---
사람들이 중고 물품을 사고 파는 중고 거래 마켓에 서로 대화를 하기 위한 1:1 채팅 기능을 추가한 중고마켓 프로젝트입니다.
